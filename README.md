Estos son dos proyectos en Python que desarrollé como ejercicios prácticos para aprender a trabajar con datos de usuario, validaciones y cálculos automáticos.

1. Calculadora de Edad

Permite ingresar una fecha de nacimiento y calcula automáticamente la edad.

Indica si la persona ya cumplió años este año o si su cumpleaños es hoy.

Maneja casos especiales como el 29 de febrero en años no bisiestos.

Incluye validaciones para asegurar que la fecha ingresada sea correcta.

2. Sistema de Evaluación de Alumnos

Permite registrar alumnos, sus calificaciones por materia y calcular su promedio.

Decide automáticamente si cada alumno aprobó o reprobó según su promedio y por materia.

Muestra un reporte completo con detalles de cada estudiante.

Valida que los campos no estén vacíos y que las calificaciones estén dentro del rango permitido (0 a 10).

Estos proyectos son un ejemplo de cómo se puede usar Python para automatizar tareas cotidianas y manejar datos de manera segura y clara.
