En este proyecto creamos un sistema básico para una tienda usando Python. La idea fue dividir cada parte importante en archivos separados para que todo se vea ordenado y sea más fácil trabajar con ello. También usamos conceptos de programación como “herencia”, “abstracción”, “composición” y “agregación”, que ayudan a organizar mejor el código y relacionar unas clases con otras. El objetivo fue practicar cómo modelar un sistema real, pero de forma simple y clara.
